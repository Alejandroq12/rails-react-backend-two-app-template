# 📖 Rails and React Backend - Webpack Two App Template. <a name="about-project"></a>

The "Rails and React Backend - Webpack Two App Template" project serves as a blueprint for developing full-stack applications. This component is the backend API, created using Ruby on Rails, designed to be integrated with the frontend found here:

[Reacts.js/Rails front-end](https://github.com/Alejandroq12/rails-react-frontend-two-app-template)

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

[Install Ruby](https://www.ruby-lang.org/en/documentation/installation/)

### Setup

```sh
cd my-folder
git clone https://github.com/Alejandroq12/rails-react-backend-two-app-template.git
```

### Install

Install all necessary dependencies listed in the `Gemfile`.

```sh
bundle install
```

### Database


```sh
sudo service postgresql start
```

Create database:

```sh
rails db:create
```

Run migrations:

```sh
rails db:migrate
```

Add the seeds:

```sh
rails db:seed
```

### Usage

To run the project, navigate to the project directory and execute the following command:

```sh
rails s
```
